Правила игры

При каждом движении чисел будет генерироваться клеточка с значением 2 или 4 ;

Числа не будут двигаться, если они упираются в стены или в неравное число ;

Нужно объединять одинаковые числа путём их столкновения при движении поля ;

Игра закончится, если не будет возможности сделать ход

Инструкция по управлению

Для управления используйте клавиши стрелочек на клавиатуре :

Для передвижения всех чисел вверх используйте клавишу ↑ или W ;

Для передвижения всех чисел вниз используйте клавишу ↓ или S ;

Для передвижения всех чисел влево используйте клавишу ← или A ;

Для передвижения всех чисел вправо используйте клавишу → или D
