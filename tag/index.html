Правила игры

Цель игры : переместить плитки таким образом, чтобы они оказались расположенными по порядку (от 1 до 15), а пустая клетка была в правом нижнем углу.

Инструкция по управлению

Для передвижения ячейки, соседней с пустой, нужно щёлкнуть на нужную ячейку с числом левой кнопкой мыши
